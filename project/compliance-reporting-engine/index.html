Compliance Reporting Engine

Client: Global retail & fashion Year: 2024 Role: AI/ML Engineer

Hybrid AI and rule‑based engine that increased compliance reporting accuracy from 40% → 80% across 5,000+ monthly transactions by automating data checks, classification, and anomaly detection.

Context

The client operated across multiple geographies and retail channels, with compliance reporting historically relying on manual review of transaction logs. Accuracy was inconsistent (40–50%), and regulatory risk exposure was high. The organisation needed an automated system to classify transactions, detect anomalies, and flag violations with confidence.

Approach

I built a hybrid engine combining rule‑based classification with AI pattern detection. The system ingests transaction data, applies deterministic rules for known compliance categories, then uses machine learning and LLM logic to flag edge cases and new risk patterns.

  • Designed a rule library capturing regulatory categories and compliance thresholds.
  • Implemented automated data cleaning and feature engineering from raw transaction logs.
  • Integrated anomaly detection (isolation forests, statistical outliers) for novel patterns.
  • Added LLM‑based reasoning to assess contextual risk and generate compliance narratives.
  • Built a dashboard for compliance teams to review and audit flagged transactions.

Impact

  • Compliance reporting accuracy improved from 40% → 80% on validation datasets.
  • Processing time reduced from weeks to hours, enabling faster regulatory responses.
  • False positive rate minimised through refined rule tuning and ensemble methods.
  • Provided audit trail and explainability for each compliance decision.

Responsibilities

Full system design: rule library development, data pipeline engineering, anomaly detection model selection, LLM integration, dashboard UX, and deployment to production.

Technologies

Python scikit-learn LLMs SQL Dashboards Rule engines

What this illustrates

How rule‑based and ML systems can work together to achieve high accuracy in regulated domains, where both deterministic compliance and adaptive anomaly detection are critical.
